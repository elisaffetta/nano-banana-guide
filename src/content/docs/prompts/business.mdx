## Советы для коммерческой фотографии

### ✅ Принципы успешной коммерческой съемки:

**Для товарных фото:**
- Четкие, резкие изображения высокого качества
- Нейтральные фоны для акцента на товаре
- Правильное освещение без резких теней
- Демонстрация всех важных деталей

**Для рекламы:**
- Яркие, привлекающие внимание цвета
- Четкий фокус на главном объекте
- Эмоциональная привлекательность
- Соответствие целевой аудитории

### 🎯 Психология коммерческих изображений:

**Цветовая психология:**
- **Красный** - энергия, срочность, акции
- **Синий** - доверие, профессионализм, технологии
- **Зеленый** - природа, здоровье, экология
- **Черный** - премиум, роскошь, элегантность

**Композиция:**
- **Правило третей** для динамичности
- **Центральная композиция** для акцента на товаре
- **Диагонали** для создания движения
- **Негативное пространство** для акцента

### 💼 Отраслевые особенности:

**E-commerce:**
- Белый фон - стандарт маркетплейсов
- Множественные ракурсы товара
- Детальные фото особенностей
- Lifestyle контент для эмоций

**B2B сектор:**
- Профессиональные портреты
- Офисные интерьеры
- Технологичность и надежность
- Командные фотографии

**Социальные сети:**
- Квадратные форматы для Instagram
- Вертикальные для Stories и TikTok
- Яркие, контрастные изображения
- UGC-стиль для аутентичности

## Технические требования

### Форматы и размеры:
- **Маркетплейсы**: 1000x1000px, белый фон
- **Социальные сети**: 1080x1080px для квадрата
- **Баннеры**: 1920x1080px или шире
- **Печать**: 300 DPI минимум

### Качество изображений:
- Резкость и четкость деталей
- Правильная экспозиция
- Естественные или точные цвета
- Отсутствие шумов и артефактов
